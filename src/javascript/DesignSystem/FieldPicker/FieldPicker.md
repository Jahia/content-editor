### Props


- *field* :
- *fieldData* :
    - *url* : url of the displayed image / icon
    - *name* : name part of the field
    - *info* : info part of the field
- *id* : uniq id 
- *editorContext* :
    - *site* : current site
    - *lang* : current content language
- *children* : the picker
